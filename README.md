# Viterbi Algorithm Implementation

## Overview
The Viterbi Algorithm is a very useful algorithm in Computational Biology, as it helps find the hidden state path with the highest probability for a given Hidden Markov Model (HMM) and observation sequence.

It utilises dynamic programming, which helps reduce the exponential time complexity involved in finding the best hidden state path.

The algorithm has been implemented in C++ and run on the input HMM provided for the assignment.

## Files

- `Viterbi.cpp`  
  Main source code where the algorithm is implemented.

- `input.txt`  
  Contains the input HMM and observation sequence.

- `output.txt`  
  Stores the generated output of the algorithm.
